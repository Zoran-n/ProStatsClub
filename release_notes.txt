## [0.4.60] - 2026-05-04
### Ajouté
- **Web Worker** : Déportation des calculs statistiques lourds (classements, agrégats) dans un thread séparé pour une UI fluide.
- **Tests Unitaires** : Couverture des heuristiques de l'IA via Vitest (détection d'anomalies, recommandations de poste).
- **Système de Logs Exportables** : Nouveau moteur de logging avec export JSON depuis l'onglet "Mon Profil".
- **Chiffrement AES-GCM** : Implémentation du chiffrement pour sécuriser les données sensibles du fichier de paramètres.
- **Mise à jour Delta** : Synchronisation incrémentale optimisée pour réduire l'usage de la bande passante.

### Corrigé
- **Nettoyage CI** : Suppression de tous les imports et variables inutilisés pour valider le build GitHub Actions.
- **Stabilité TypeScript** : Correction des erreurs d'itérations sur les données de l'API EA.
